# Image slots

Drop your own licensed photography into this folder using these exact
filenames and every corresponding section will pick it up automatically —
no code changes required. Until an image is added, each section shows a
branded illustrated fallback instead of a broken image.

- `hero-car-carrier.jpg` — Hero background. A loaded car carrier on a
  Nigerian road works best; wide/landscape, at least 1800px wide.
- `services/transportation.jpg` — Vehicle Transportation service card.
- `services/enclosed.jpg` — Enclosed & High-Value Haul service card.
- `services/showcase-carrier.jpg` — "Car Carrier in Action" showcase panel (home page).
- `services/showcase-driven.jpg` — "Driven With Care" showcase panel (home page).
